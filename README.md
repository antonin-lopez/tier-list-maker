# 🏆 AI Tier List Maker

Un outil de création de Tier Lists ultra-léger, **entièrement conçu par Intelligence Artificielle**, pour ceux qui veulent de l'efficacité sans fioritures.

## 🚀 Pourquoi ce projet ?

Ce projet est né d'un constat simple : la plupart des sites de Tier Lists demandent une inscription, imposent des publicités ou ne permettent pas de sauvegarder/charger ses données de manière simple et privée. 

**L'objectif ici :**
* **Zéro inscription :** Pas de compte, pas de tracking.
* **Zéro dépendance :** Tu possèdes tes données sous forme de fichier JSON.
* **Sur mesure :** Créé pour répondre exactement à mes besoins techniques.

> [!IMPORTANT]
> **Projet 100% IA :** Tout le code de cette application a été généré via une collaboration entre l'utilisateur et une Intelligence Artificielle (Gemini/Claude).

---

## ✨ Fonctionnalités

* **Customisation totale :** Ajoute, supprime, renomme et change la couleur de tes Tiers (S, A, B...).
* **Éléments mixtes :** Supporte aussi bien les **images** (via upload local) que les **étiquettes textuelles**.
* **Drag & Drop fluide :** Conçu avec `@dnd-kit` pour fonctionner aussi bien à la souris sur **PC** qu'au doigt sur **Mobile**.
* **Export/Import JSON :** Télécharge ta Tier List pour la reprendre plus tard ou la partager. Tout est stocké dans le fichier (y compris les images converties en Base64).
* **Responsive Design :** Interface moderne et sombre (Dark Mode) qui s'adapte à tous les écrans.
